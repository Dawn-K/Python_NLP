```html
开始标签常被称为开放标签（opening tag），结束标签常称为闭合标签（closing tag）。
HTML 标签对大小写不敏感

<!--作为整体的:注,以下全是成对出现-->
<body>
<title>
<h1> ~ <h6>   之间的文本被显示为标题
<legend>      也是标题
<summary>     和上文的标题功能相似
<p>           之间的文本被显示为段落,一般是整体
<textarea>    表示一段文本
<noframes>    为那些不支持框架的浏览器显示文本
<noscript>    定义在脚本未被执行时的替代内容（文本）
<option>      定义下拉列表
<caption>
<dl>,<dt>     表示表格(行列?)其中的一般是整体,一般不会插到某句话中间
<li>,<th>,<td>和上个差不多
<center>      定义居中信息
<address>     定义联系信息
<aside>       定义 article 以外的内容
<blockquote>  表示长段引用,一般一定在句子两端
<q>           表示短的引用,一般出现在句子的两端
<a>           定义一个超链接  
<command>     和<a>差不多,不过是定义按钮
<audio>       定义一个音频按钮


<!--只会在句中出现-->
<bdi> </bdi>几乎只会在句子之中出现

<!-- br标签特别喜欢插入到文本中间,而且几乎只插入到文本中间或者单独出现,和文本不接触,而且br一定成单出现-->
<br/> 标签：在不产生一个新段落的情况下进行换行（新行）
// <spac>
<p>This is<br/>a para<br/>graph with line breaks</p>
<br/> 元素是一个空的 HTML 元素。由于关闭标签没有任何意义，因此它没有结束标签。


<!--下面列出的是既可以标记部分文本,也可以标记整个文本-->
<font size="3" color="red"></font> 定义文本
<b>     定义粗体文本
<big>   定义大号字
<small> 定义小号字
<em>    定义着重文字
<mark>  突出显示
<del>   定义删除字
<s>     定义删除字
<strike>定义删除字
<m>     突出显示
<i>     定义斜体字
<strong>定义加重语气
<sub>   定义下标字
<sup>   定义上标字
<span>  通过样式来格式化它们
<ins>   定义插入字
<bdo>   设置文字方向
<u>     为文本添加下划线
<cite>  参考文献(一般是参考文献的标题)
<time>  标签定义日期或时间,例子:<time datetime="2008-02-14">情人节</time>



<!--此外,以下的部分大多涉及到特定的代码,应该不翻译(感觉此部分不用特意处理,翻译人员会自行跳过此部分)-->
<code>      代码片段,感觉应该跳过,里面的内容不应该翻译
<kbd>       定义键盘码。
<samp>      定义计算机代码样本。
<tt>        定义打字机代码。
<var>       定义变量。
<pre>       定义预格式文本。
<listing>   不赞成使用。使用 <pre> 代替。
<plaintext> 不赞成使用。使用 <pre> 代替。
<xmp>       不赞成使用。使用 <pre> 代替。
<style type="text/css">
h1 {color:red}
p {color:blue}
</style>    中间的部分应该完全保留

特殊
<dfn>  里面往往包含术语(命名实体识别?)
<abbr> 里面包含缩写,应当是不翻译
    // xiaolaoshi  copy model
```
